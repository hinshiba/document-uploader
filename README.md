# 講義ドキュメントマネージャ

## 解決する課題

- 現状の Google Drive 共有では名前が統一されていない
- だれでも削除できる

## 機能

機能は実装中です

- ファイルのアップロード/一覧/絞り込み/ダウンロード
- 教科一覧
- ファイルのサーバ側検証
- アップロード者の記録
- 認証: 外部ネットワークからのアクセス

## 技術構成

予定を含む場合があります

- **フロントエンド**: HTML, CSS, TS
- **バックエンド**: Rust(Axum)
- **ファイル保存**: ローカルファイルシステム
- **バックアップ**: Google Drive
- **認証**: Cloudflare Tunnelに付属のCloudflare Access
