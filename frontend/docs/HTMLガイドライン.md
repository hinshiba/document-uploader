# HTMLガイドライン

## 構造

- `<body>`は次の構造を順にとる
  1. `<header> | <aside>`: 主なページ間移動を提供
  2. `<main>`: 主となるコンテンツ
  3. `(<section> | <article>)*`: 主でないコンテンツ
  4. `<footer>`

- `<header>`または`<aside>`には，`<nav>`を含むこと
- `<main>`, `<section>`, `<article>`には可視要素として見出しを最初に含むこと

## 構文

- 空要素に終止タグを用いないこと
  - 自己完結型タグはHTMLには存在しない

## 各タグについて

### nav, aside

- 原則文書内に1つにすること

### label

- 対応する要素の前に配置すること
- 暗黙的なラベルの定義は用いないこと

### div

- `<section>`, `<aritcle>`等が不適切な場合のみ用いること
- 内部を空にしないこと
  - HTMLと要求仕様の制限上不可避である場合は除く
- `<p>`, `<h2>`等の子要素として配置しないこと
  - フレージングコンテンツを要求するタグの子要素としては配置できない
